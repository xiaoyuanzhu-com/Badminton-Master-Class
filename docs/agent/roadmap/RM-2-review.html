🔄 架构转型：Content-as-Code
RM-2 最大的变化是内容即代码（Content-as-Code）架构转型。 管理后台从 CRUD 编辑器转为只读 Web 客户端，内容管理完全迁移到 Git 仓库中的 JSON 文件。 带来三大收益：可审查、可 diff、Agent 可直接写入。
